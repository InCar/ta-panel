# 介绍

一个使用 `vite` + `vue3` + `pinia` + `Element +` + `typescript` 完整技术路线开发的项目，需要Node.js 版本14.18+，16+

# 推荐开发工具

 开发工具 vscode + Volar 组合(注意：Volar 取代了我们之前为 Vue 2 提供的官方 VSCode 扩展 Vetur。如果你之前已经安装了 Vetur，请确保在 Vue 3 的项目中禁用它。)


# yarn 

1. 安装依赖
    ```sh
    # 推荐使用 yarn
    yarn
    ```

2. 启动项目
    ```sh
    yarn dev
    ```

3. 部署
    ```sh
    yarn build
    ```
