# 介绍

  数据分析平台

# Dependencies
+ [nodejs 16.10+](https://nodejs.org)
+ [yarn  1.22.19+](https://classic.yarnpkg.com/)

# 标准开发工具

 开发工具 vscode + Volar 组合(注意：Volar 取代了我们之前为 Vue 2 提供的官方 VSCode 扩展 Vetur。如果你之前已经安装了 Vetur，请确保在 Vue 3 的项目中禁用它。)


# Build & Run

1. 安装依赖
    ```sh
    yarn
    ```

2. 构建应用
    ```sh
    yarn build
    ```


3. 本地调试项目
    ```sh
    yarn dev
    ```
